Поднять простое приложение. Убедиться, что базовый тест пройден.
----------

Сделан простой контроллер с тремя ednpoint'ами:
/ - доступен всегда
/user - доступен пользователям с ролями USER и ADMIN
/admin - доступен только пользователям с ролью ADMIN.

С помощью spring-security была подключена авторизация (из коробки), добавлены in-memory пользователи: user и admin с соответствующими ролями.
